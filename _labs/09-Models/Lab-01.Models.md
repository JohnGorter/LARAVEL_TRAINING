# Lab Databases
## Lab time: 40 minutes

## Excercise 1: make a migration to create a Gebruikers table
Use a migration to make a Gebruikers table that has at least:
- an unique email adress
- firstname
- lastname
- age
Make use of the appropriate data types for each of the fields

## Excercise 2: make a database seeder to seed at least 3 records
Use a database seeder to seed your Gebruikers table. Add at least 3 new valid records to the Gebruikers table using your seeder. 

## Excercise 3: use a Gebruikers records in a view
Change a route or controller to query the database for the Gebruikers
and display the records in the view to the user.
